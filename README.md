# Before 404 Archive

本仓库用于存档那些**有可能随时被删除、DMCA 下架或遭到审查**的公开资料，包括但不限于大厂源码泄漏、敏感文件、争议性事件相关材料等。

> "If it can be deleted, it should be archived."

## 收录标准

- 已公开流传、但原始来源存在被下架风险的资料
- 涉及科技公司源码泄漏事件
- 具有重要历史意义或公众知情价值的文件
- 曾经或正在面临 DMCA / 政府要求删除 / 平台审查压力的内容

## 免责声明

本仓库所有内容均来源于互联网公开流传的资料，存档目的为**历史记录与信息自由**，不代表对任何当事方的立场背书。若您认为某项内容侵犯了您的合法权益，请提交 Issue 说明情况。

## 贡献

欢迎提交 PR 补充新的存档事件，请在 PR 描述中注明：
- 事件名称与日期
- 资料来源（原始链接，即使已失效）
- 为何存在被删除的风险
